# Google Colab Templates

These notebooks are reviewer-friendly templates for demonstrating the full technical pipeline.

- `colab-template-01-data-ingestion.ipynb`: source ingestion from CSO, Central Bank, and World Bank scripts.
- `colab-template-02-signal-pipeline.ipynb`: transform and scoring pipeline walkthrough.
- `colab-template-03-retrieval-demo.ipynb`: API retrieval and Flask orchestration behavior.

## How to use

1. Upload to Colab or open from GitHub with the provided Colab badge links.
2. Fill Azure credential environment variables before running ETL/API cells.
